# ML (aprendizaje automático)

Aquí se colocarán scripts y datos para preprocesado, entrenamiento y evaluación de modelos.

Estructura sugerida:

- `ml/data/` - datasets crudos y preparados
- `ml/preprocess.py` - script para preparar los datos
- `ml/train.py` - script de entrenamiento
- `ml/models/` - modelos entrenados

Archivo placeholder `preprocess.py` se puede agregar cuando comencemos a convertir contribuciones aprobadas en dataset.
